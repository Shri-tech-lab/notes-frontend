# 📝 Notes App

A clean and responsive Notes App built using HTML, CSS, and JavaScript.  
This application allows users to create, edit, save, and delete notes easily with a simple user-friendly interface.

## 🚀 Live Demo

🔗 [Click Here to Use Notes App](https://shri-tech-lab.github.io/notes-frontend/)

---

## 📌 Features

- 📝 Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 💾 Auto save notes
- 📱 Responsive design
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

```bash
notes-frontend/

├── index.html

```

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/shri-tech-lab/notes-frontend.git
```

2. Open project folder

```bash
cd notes-frontend
```

3. Open `index.html` in browser
